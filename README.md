# 📡 Enhancing Campus Security Using IoT

## 📖 About the Project

This repository presents my published research paper, *"Enhancing Campus Security Using IoT"*. The project proposes a smart campus security system that leverages IoT technologies to improve safety through real-time monitoring, emergency alerts, and mobile-based surveillance.

## 🎯 Objectives

- Improve campus security using IoT devices
- Enable real-time monitoring and surveillance
- Detect fire and abnormal noise
- Send instant alerts to users
- Provide remote access through a mobile application

## 🛠️ Technologies Used

- Flutter
- Dart
- Firebase Realtime Database
- ESP32-CAM
- Arduino IDE
- IoT Sensors
- Wi-Fi Communication

## ✨ Key Features

- Live Camera Monitoring
- Fire Detection
- Noise Detection
- Real-time Notifications
- Firebase Cloud Integration
- Mobile Application
